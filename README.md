Odoo Forked
----
This repository contains feature extensions for mainly Odoo:16.0.0.

Running in Docker
-------------------------
- Execute the below command:
  ```docker-compose up -d```
  
Testing
-----------------
- Execute the below command
  ```python -m ...addons.od_product.tests.test_extended_product_template```
  
  
Requirements
----------------
- See the [here](https://hub.docker.com/_/odoo/)
